## shopping
인터넷 상점의 주요 기능을 구현한 '마일리지 상점'을 제작하는 프로젝트
구매자에게 1,000,000 포인트를 지급하고 포인트 한도 내에서 상품을 구입할 수 있다.
이 프로그램은 8가지 기능을 가지고 있다.
1. 상품 확인
2. 상품 구매
3. 잔여 마일리지 확인
4. 배송 상태 확인
5. 장바구니 확인
6. 장바구니 초기화
7. 장바구니 상품 주문
8. 프로그램 종료

그리고 구입할 수 있는 상품은 5가지가 있다.
| 상품 | 포인트 | 수량 |
|--|--|--|
| 텀블러 | 10000 | 30 |
| 내셔널지오그래픽 롱패딩 | 500000 | 20 |
| 디스커버리 백팩 | 400000 | 20 |
| 나이키 운동화 | 150000 | 50 |
| 빼빼로 | 1200 | 500 |
